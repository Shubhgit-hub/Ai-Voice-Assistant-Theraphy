# 🎙️ VoiceRx — AI Voice Therapy Assistant

VoiceRx is an AI-powered speech therapy and pronunciation training application that helps users improve speech clarity, pronunciation accuracy, articulation, and speaking confidence through real-time voice analysis and intelligent feedback.

Built with Python, Streamlit, OpenAI Whisper, Librosa, and Google Text-to-Speech, the application provides an interactive environment for practicing speech and tracking improvement over time.

## ✨ Features

* 🎤 Record speech directly inside the application
* 🧠 Speech-to-text transcription using OpenAI Whisper
* 📊 Pronunciation accuracy scoring
* 🔍 Pitch and voice energy analysis
* 📈 Waveform and spectrogram visualization
* 🎯 Adaptive speech exercises
* 🗣️ Tongue twister practice mode
* 🤖 AI-inspired speech coaching feedback
* 🔊 Text-to-Speech exercise playback
* 📋 Session tracking and progress monitoring
* 🌙 Modern clinical dark-themed interface

## 🛠️ Tech Stack

* Python
* Streamlit
* OpenAI Whisper
* Librosa
* NumPy
* Pandas
* Matplotlib
* SoundDevice
